⚙️ Google Sign-In belum diaktifkan di Supabase
  1. Buka supabase.com/dashboard → pilih project Anda
  2. Klik Authentication → Providers → cari Google
  3. Aktifkan toggle Google, lalu isi Client ID & Client Secret dari Google Cloud Console
  4. Di Google Cloud Console: APIs → Credentials → OAuth 2.0 → tambah https://[project].supabase.co/auth/v1/callback sebagai Authorized redirect URI
  5. Klik Save di Supabase, lalu coba lagi ✅
